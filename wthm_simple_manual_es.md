# Especificaciones Técnicas del Producto

## Rango de Medición y Precisión

<div class="specs-container">
<div class="specs-column">
<ul>
<li>Rango de Temperatura: -25 ~ +65 ℃</li>
<li>Precisión: ±0,4°C (rango 0~65°C), ±0,8°C (rango completo)</li>
</ul>
</div>
<div class="specs-column">
<ul>
<li>Rango de Humedad Relativa: 0 ~ 100 %HR</li>
<li>Precisión: ±3,5% (rango 10~90%), ±5% (rango completo)</li>
</ul>
</div>
</div>


## Fuente de Alimentación

- Soporta entrada USB Tipo-C 5V. Corriente nominal: 100mA, Corriente pico: < 200mA.
- Soporta alimentación por terminal DC interna, voltaje de entrada nominal DC12V, rango DC9~28V.


## Comunicación Inalámbrica WLAN

Soporta redes Wi-Fi IEEE 802.11 b/g/n banda de 2,4 GHz.


# Descripción de la Pantalla y Teclas

- En el modo de visualización normal, la pantalla LCD del dispositivo muestra la temperatura, la humedad y otros valores de parámetros.
- Mantenga presionada la tecla √ durante 1 segundo para acceder a la interfaz de parámetros del dispositivo, donde puede ver la dirección IP del dispositivo, la hora, la dirección MAC, la red Wi-Fi, la versión del firmware, etc.
- Mantenga presionadas simultáneamente las teclas ↓ y ↑ del medio durante 5 segundos para acceder a la interfaz de configuración Wi-Fi. Escanee el código QR en la pantalla con un teléfono o tableta para acceder al programa de configuración Wi-Fi.
- Mantenga presionadas simultáneamente las teclas laterales ⟲ y √ durante 5 segundos para acceder a la interfaz de restablecimiento de fábrica.
